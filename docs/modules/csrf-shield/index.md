# @fehujs/csrf-shield

This module provides a CSRF protection system to secure your forms.

## Module configuration

- ``TOKEN_COOKIE_NAME: "csrf_token"``
- ``TOKEN_HEADER_NAME: "x-csrf-token"``
- ``TOKEN_BODY_NAME: "_token"``
- ``TOKEN_EXPIRES: "300000"``
