# @fehujs/sessions

Fehujs sessions handling package.

!!! warning
    At this time, only the session id handling is implemented.

## Module configuration

- ``ID_COOKIE_NAME: "session_id"``
- ``ID_COOKIE_EXPIRES: 1800000``
- ``ID_LENGTH: 16``
